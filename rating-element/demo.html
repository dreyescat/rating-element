An example of using <rating-element>:

Default rating

<rating-element></rating-element>

Use fontawesome icons

<rating-element empty="fa fa-star-o" filled="fa fa-star"></rating-element>

Use fontawesome icons with custom size

<rating-element empty="large fa fa-heart-o" filled="large fa fa-heart"></rating-element>

Read only rating with a value

<rating-element readonly value="2"></rating-element>

Use custom icon

<rating-element empty="custom-symbol custom-symbol-empty" filled="custom-symbol custom-symbol-filled"></rating-element>

Custom start and stop rate [1..10)

<rating-element start="1" stop="10"></rating-element>

Custom start and stop rate [1..10) with step 2

<rating-element start="1" stop="10" step="2"></rating-element>

Custom stop rate [0..-10) with step -2

<rating-element stop="-10" step="-2"></rating-element>
