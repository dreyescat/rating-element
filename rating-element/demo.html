An example of using <rating-element>:

Default rating

<rating-element></rating-element>

Use fontawesome icons

<rating-element empty="fa fa-star-o fa-2x" filled="fa fa-star fa-2x" symbols="../bower_components/fontawesome/css/font-awesome.css"></rating-element>

Use typicons icons

<rating-element empty="typcn typcn-thumbs-down large" filled="typcn typcn-thumbs-up large" symbols="../bower_components/typicons/src/font/typicons.css"></rating-element>

Use fontello generated icons

<rating-element empty="icon-meh large" filled="icon-smile large" symbols="../dist/fontello/css/fontello.css"></rating-element>

Use bootstrap CDN glyphicons

<rating-element empty="glyphicon glyphicon-heart-empty large" filled="glyphicon glyphicon-heart large" symbols="//netdna.bootstrapcdn.com/bootstrap/3.0.0/css/bootstrap-glyphicons.css"></rating-element>

Read only rating with a value

<rating-element readonly value="2"></rating-element>

Use custom icon

<rating-element empty="custom-symbol custom-symbol-empty" filled="custom-symbol custom-symbol-filled"></rating-element>

Custom start and stop rate [1..10)

<rating-element start="1" stop="10"></rating-element>

Custom start and stop rate [1..10) with step 2

<rating-element start="1" stop="10" step="2"></rating-element>

Custom stop rate [0..-10) with step -2

<rating-element stop="-10" step="-2"></rating-element>
